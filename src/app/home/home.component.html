WBS Project:
Predicting Medicine Entities and RDF Triple Visualization

This project aims to predict medicine entities using a machine learning model and construct and visualize RDF triples. The project utilizes data science techniques and semantic web technologies to achieve these objectives.

Features
  • Machine learning model for predicting medicine entities
  • Construction of RDF triples
  • Visualization of RDF triples
Technologies Used
  • FastAPI (Python)
  • PyTorch (Python)
  • RDFlib (Python)
  • Angular (TypeScript)
  • Bootstrap (CSS Framework)
  • @swimlane/ngx-graph (Visualization library)
